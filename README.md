# J.A.R.V.I.S
## Use
The use of J.A.R.V.I.S is very simple, you must use chrome browser, also you can try other browser but I don`t ensure that works correctly
### 1ºStep: 
You must clone the repository.
### 2ºStep: 
You must enter in the repository folder.
### 3ºStep:
You must have installed node previously.
### 4ºStep:
You must do the following in the terminal: node server.js.
### 5ºStep:
You must go to the chrome browser and write the URL: localhost:3000.
### 6ºStep:
To talk with J.A.R.V.I.S, you must push the < key in your keyboard and talk. If you want interrupt J.A.R.V.I.S, you must push - key.
